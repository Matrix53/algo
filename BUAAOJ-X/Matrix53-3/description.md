## 题目描述

且说道那 Matrix53 不仅下棋胜了 Alex，还顺利地在规定时间内统计出了数列中的三角形的总数，扬名四方。

数列大师 Marvolo 听说了 Matrix53 的事迹，于是决定和他共同研究一个数列问题，他给 Matrix53 发了一封信：

你能帮助 Matrix53 和 Marvolo 解答出这道题吗？

## 输入

第一行为数列的长度 n

第二行为 n 个正整数，代表数列中的元素 $a_i$

## 输出

输出一个整数，表示这个数列中的三角形的总数

## 输入样例

    4
    1 2 2 2

## 输出样例

    4

## 数据范围

$3\leq n \leq 10^3$

$1\leq a_i \leq 10^7$
