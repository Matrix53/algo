## 题目描述

给定 $n$ 个点和 $m$ 条有向边，边具有容量。

给定 $p$ 个源点和一个汇点 $t$，求源点到汇点的最大流。

## 输入

第一行四个正整数$n \ m \ p \ t$，点的序号从$1$到$n$

接下来$m$行，每行三个正整数$u \ v \ c$，表示$u$到$v$具有容量为$c$的有向边

接下来一行，$p$个正整数，表示源点集

## 输出

输出源点到汇点的最大流

## 输入样例

    3 2 2 3
    2 3 3
    1 3 2
    1 2

## 输出样例

    5

## 数据范围

$3\leq p ＜ n \leq 100$

$1\leq m \leq 5000$

$0\leq c \leq 2^{31}-1$
